# Mayur Varshney — Portfolio

Personal portfolio built with React + TypeScript + Vite. Showcases 9+ years of
experience as a Senior Software Developer / Team Lead — React Native, React,
Node.js, and AI-augmented development (Claude Code, GitHub Copilot).

**Live site:** https://YOUR_USERNAME.github.io/portfolio/

## Run locally

```bash
npm install
npm run dev
```

## Lint

```bash
npm run lint
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages

This repo auto-deploys via GitHub Actions (`.github/workflows/deploy.yml`)
on every push to `main`. Just make sure Pages is set to "GitHub Actions" as
the source in repo Settings → Pages.
